Projeto Future4

Uma web page simples feita para o processo de seleção da escola Future4 (atual Labenu). Além de espelhar uma imagem fornecida via PDF, o projeto deveria ser responsivo utilizando como exemplo um iPhone X, além de conter um campo, contando com um validador de e-mail, para a inserção do e-mail de um possível usuário.

Tecnologias utilizadas:

  • HTML
  • CSS
  • JavaScript
