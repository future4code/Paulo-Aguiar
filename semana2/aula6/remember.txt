2. Block ocupa todo o espaço horizontal onde se encontra o elemento. Inline ocupa apenas o espaço 
horizontal que o elemento precisa.
Block: 
<p> para fazer um novo parágrafo
<h1> para criar um título
<li> para criar um novo item de uma lista
Inline:
<span> para criar uma frase, ou palavra, uma ao lado da outra
<label> para criar um nome para o campo a ser preenchido
<input> para criar um campo a ser preenchido, ou um botão

3.

c. div>p {
   color: green;
   }
   div>p:first-child {
   color: yellow;
