2. 
&&: é um operador que só retorna verdadeiro se ambas afirmações forem verdadeiras.
    Ex.: false && true = false; true && true = true. 
||: é um operador que só retorna falso se ambas afirmações forem falsas.
    Ex.: false || true = true; false || false = false. 
!: é um operador que inverte a afirmação lógica.
    !false = true; !true = false.
